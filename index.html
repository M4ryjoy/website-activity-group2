Terminologies

Java

Java is an object-oriented programming language used in back-end development for web applications, enterprise software, and mobile apps. It is one of the most popular languages for server-side programming.

Firebase

Firebase is a backend-as-a-service (BaaS) platform by Google that provides a real-time database, authentication, cloud functions, and other services for web and mobile development.

Node.js

Node.js is an open-source runtime environment that allows JavaScript to run on the server side, making it a popular tool for back-end development.

MySQL

MySQL is an open-source relational database management system (RDBMS) used in the back end to store and manage structured data.

Vue.js

Vue.js is a JavaScript framework used for front-end development, but it can also be used with Node.js on the back end to build server-side rendered applications.

Jupyter Notebook

The Jupyter Notebook is an open-source web application that enables developers to create and share documents containing live code, equations, visualizations, and narrative text. While it is commonly used in data science and machine learning, it is also utilized in back-end development for data analysis, prototyping, and code documentation.

JSON

JavaScript Object Notation (JSON) is a lightweight data format used for sending and receiving data between the client and server. It is commonly used in REST APIs.

Backbone.js

Backbone.js is a JavaScript library that provides structure to web applications by offering models with key-value binding and custom events. While it is commonly used on the front-end, it can also be utilized on the back-end to manage application data and logic.

Redis

Redis is an open-source in-memory data structure store used as a database, cache, and message broker in back-end development.

REST API

The REST API (Representational State Transfer Application Programming Interface) is an architectural style used for sending and receiving data between the client and server.
